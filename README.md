# ComplexLayout
Android Complex Layout
效果如下图所示，由一个ListView+ScrollVie+LinearLayout组成：

本项目解决的问题主要是一个界面中存在ListView和其他的View时，同时下拉刷新时的效果是整体下拉而非ListView单独下拉。<br>
ListView的head采用了MD效果，个人比较喜欢...
